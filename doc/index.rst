Welcome to *theragenome*'s documentation
========================================

The *theragenome* package contains all the necessary tools to reproduce the data pipeline of the Theranostic Genome project.

Check out the :doc:`Introduction <intro>` section for general information concerning the package, the :doc:`Installation <install>` section for instructions to install the software and the :doc:`Usage <usage>` section for information on how to use the library.
